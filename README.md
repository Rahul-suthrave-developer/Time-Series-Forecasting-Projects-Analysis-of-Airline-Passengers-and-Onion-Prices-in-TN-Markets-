# 📈 Time Series Forecasting Projects Analysis of Airline Passengers and Onion Prices in TN Markets

## 📌 Repository Overview

This repository contains multiple **Time Series Analysis and Forecasting projects** implemented using Python.

The projects demonstrate techniques such as:

* Trend Analysis
* Seasonal Decomposition
* Statistical Forecasting
* Deep Learning for Time Series

The goal of this repository is to showcase practical applications of **time series analysis in real-world datasets**.

---

# 📂 Projects Included

## 1️⃣ Crude Oil Import & Export Forecasting

Forecasting India's crude oil import and export trends using statistical and deep learning models.

### Techniques Used

* ARIMA
* SARIMA
* GARCH
* LSTM
* CNN + LSTM Hybrid Models

### Applications

* Energy demand forecasting
* Economic planning
* Oil market trend analysis

---

## 2️⃣ Airline Passenger Time Series Analysis

Analysis of airline passenger data to understand long-term travel demand trends.

### Techniques Used

* Time Series Visualization
* Seasonal Decomposition
* Trend Analysis

### Applications

* Airline demand forecasting
* Transportation planning

---

## 3️⃣ Onion Market Price Analysis (Tamil Nadu)

Time series analysis of onion market prices across TN markets.

### Techniques Used

* Trend Analysis
* Seasonal Decomposition
* Market price fluctuation analysis

### Applications

* Agricultural price monitoring
* Market trend analysis

# ⚙️ Installation

```
git clone https://github.com/Rahul-suthrave-developer/Time-Series-Forecasting-Projects-Analysis-of-Airline-Passengers-and-Onion-Prices-in-TN-Markets-.git
```

Navigate to the project directory:

```
cd Time-Series-Forecasting-Projects-Analysis-of-Airline-Passengers-and-Onion-Prices-in-TN-Markets-
```

Install required libraries:

```
pip install -r requirements.txt
```
